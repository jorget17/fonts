Updates to Existing Fonts

Every font that was previously shipped on Font Folio has been updated. Most of these are very minor changes that are not expected to cause any reflow of existing documents. In a few cases more extensive changes have been made.

Most notably, the coding of OpenType "fraction" and "ordinals" support has been updated so that more fonts support full arbitrary fractions and more flexible use of ordinals. Note that as recommended previously, you should not apply "fraction" or "ordinal" formatting globally to text, but only to the fractions or ordinals themselves. Applying such formatting more broadly may cause problems when switching typefaces or using an updated version of the same font.

For more details on changes to specific families, see the font-specific readmes, which are available online at http://www.adobe.com/type/family_readmes.html.

Updated Type Reference Guide

The Type Reference Guide was updated to include the new typefaces on Font Folio 11. In addition, the layout was redesigned, additional language coverage icons were added, and more material was translated into French and German.
